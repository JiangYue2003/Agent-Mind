# EchoMind Wiki 文档中心

EchoMind 当前不仅支持智能客服对话、RAG 知识库和多 Agent 编排，也已经支持端到端评测能力。评测入口为 `POST /eval/run`，覆盖意图识别准确率、Agent 回复质量、LLM-as-Judge 多维评分、回归检测和优化建议生成。阅读任意文档时，都应把“可评测、可回归、可持续优化”视为本项目的重要工程能力。
